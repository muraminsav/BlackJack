Black♠️Jack

This project is a coding assignment for a Codeworks application, the objective of this project is to create a simplified version of the popular Blackjack cardgame using HTML, CSS, JacaScript and JQuery. In this game the user is able to play multiple games against the bank keeping score tally.

The value of cards two through ten is their pip value (2 through 10). Face cards (Jack, Queen, and King) are all worth ten. Aces can be worth one or eleven. When dealt a hand with an ace valued at eleven, a second ace will be valued at one to prevent the hand from exceeding twenty-one.

The goal of the game is to beat the bank by having the sum total of your cards surpass the dealers hand but not exeeding twenty-one.
